# mind2slide
Вы структурируете свои мысли при помои ментальной карты (mindmap), на основе которой будет построена презентация.
Используемые библиотеки
* построение и отображение ментальной карты [jsMind](https://jsmind.online/#sample)
  показ ментальной карты в виде последовательности слайдов [RevealJS](https://revealjs.com/)
